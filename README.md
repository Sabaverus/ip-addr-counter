# Счетчик ИП адресов

Решение задачи, условия которой находятся ниже.
Сохраняет флаг уникальности IP адреса как один бит в массиве битов в реализации BitSet.
Итого: максимальный размер коллекции ~550mb, на 100млн записей уходит около 90(±3с.) секунд при чтении напрямую из архива с текстовым файлом.
Конфигурация разработки - Ryzen 1700, SSD 970 EVO Plus, DDR4

Условия задачи:

Дан простой текстовый файл с IPv4 адресами. Одна строка – один адрес, примерно так:

```
145.67.23.4
8.34.5.23
89.54.3.124
89.54.3.124
3.45.71.5
...
```

Файл в размере не ограничен и может занимать десятки и сотни гигабайт.

Необходимо посчитать количество __уникальных адресов__ в этом файле, затратив как можно меньше памяти и времени. Существует "наивный" алгоритм решения данной задачи (читаем строка за строкой, кладем строки в HashSet), желательно чтобы реализация была лучше этого простого, наивного алгоритма.
